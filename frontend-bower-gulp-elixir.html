Frontend : Bower, Gulp & Elixir

Prerequisites.

To install all these packages you need to have node.js installed (to use npm).

        
          $ sudo apt-get update
          $ sudo apt-get install nodejs

Bower.js | A package manager for the web.

Bower works by fetching and installing packages from all over, taking care of hunting, finding, downloading, and saving the stuff you’re looking for. Bower keeps track of these packages in a manifest file, bower.json. How you use packages is up to you.

Now you can instal bower.js globally:

$ sudo npm install -g bower

That’s it. Now, you have two ways to install new packages:

  • 1. Install a needed package in terminal: 
    $ bower install {package_name} --save
  • 2. Add package information (name, version) to bower.json dependencies and update bower: 
                
              $ "dependencies": { ..., "package_name": "*" }
              $ bower update

For more look the official bower website: bower.io

Gulp.js | Automate and enhance your workflow.

Gulp is a streaming build system, by using node’s streams file manipulation is all done in memory, and a file isn’t written until you tell it to do so. You just tell gulp which files and in witch order must be compiled/concat/minified and that's it. It all works automatically.

To install gulp.js you need node.js installed (to use npm command)

        
          $ sudo apt-get update
          $ sudo apt-get install nodejs

Now you can install gulp.js globally:

        
          $ sudo npm install -g gulp
          $ npm install

To compile and concat all needed file just run:

        
          $ gulp              
          $ gulp --production (to minify)
